# Paving Calculator

A mobile-first web application for in-field asphalt paving calculations. Designed for paving crew members to quickly compute spread rates, tack rates, remaining distance, and tonnage estimates while on the job.

## Features (Planned)

- **Spread Rate Calculator** — Calculate lbs/SY from placed tons, distance, and width
- **Remaining Asphalt Distance** — Estimate how far remaining loads will go
- **Tack Rate Calculator** — Gallons needed for tack coat application
- **Tonnage Estimator** — How much material to order
- **Stick Check Height** — Quick desired height × 1.2 conversion
- **Reference Tables** — GDOT spec tables for lift thickness, mix types, and application rates

## Target Platform

- Mobile-first PWA (Progressive Web App)
- Works offline (all calculations are client-side)
- Touch-friendly large inputs for gloved hands
- High contrast for outdoor visibility
